# ISO 16269-6 统计容限区间：过程、推导、文献与应用

## 1. 标准要解决的问题

ISO 16269-6 讨论的是统计容限区间（statistical tolerance interval），不是普通的均值置信区间，也不是单个未来观测值的预测区间。

它要构造一个由样本计算出来的随机区间，使该区间以给定置信度覆盖总体中至少给定比例的个体：

```text
P_sample{ P_population(X 落在区间内) >= p } = γ
```

其中：

- `p`：内容比例或覆盖比例，例如 0.90、0.95、0.99。
- `γ = 1 - α`：置信度，例如 0.95。
- `α`：区间未能覆盖至少 `p` 比例总体的风险。

例如，“95/99 上容限限”通常表示：以 95% 的置信度，至少 99% 的总体值不超过这个上限。

ISO 16269-6:2005 已被 ISO 16269-6:2014 替代。2014 版仍然覆盖一侧和两侧统计容限区间、正态参数法和分布无关法，并增加了多个正态样本且共同未知方差的情形。

## 2. 标准中的基本程序

### 2.1 输入条件

使用前先确定：

```text
p       = 要覆盖的总体比例
γ       = 置信度，即 1 - α
n       = 样本量
side    = 单侧上限、单侧下限或双侧区间
method  = 正态参数法或分布无关法
```

样本观测值应独立。分布无关法还要求总体分布连续。

### 2.2 正态参数法

若可认为特性值来自正态总体：

```text
X ~ N(μ, σ²)
```

取样本：

```text
x1, x2, ..., xn
```

计算：

```text
x̄ = 样本均值
s  = 样本标准差
```

单侧上容限限：

```text
U = x̄ + k s
```

单侧下容限限：

```text
L = x̄ - k s
```

双侧容限区间：

```text
[x̄ - k s, x̄ + k s]
```

其中 `k` 是容限因子，取决于：

```text
k = k(n, p, γ, side)
```

标准通常以表格或附录形式给出 `k` 值。工程软件和统计软件则通过非中心 t 分布、数值积分或近似公式计算。

### 2.3 分布无关法

若不想假设正态分布，只假设总体分布连续，则使用阶次统计量。

将样本排序：

```text
X(1) <= X(2) <= ... <= X(n)
```

单侧上限可以取某个阶次统计量：

```text
U = X(s)
```

单侧下限：

```text
L = X(r)
```

双侧区间：

```text
[X(r), X(s)]
```

选择 `r`、`s`，使该区间覆盖至少 `p` 比例总体的概率不低于 `γ`。

## 3. 正态单侧容限限的精确推导

以单侧上容限限为例：

```text
U = x̄ + k s
```

希望：

```text
P_sample{ P_population(X <= U) >= p } = γ
```

因为总体正态：

```text
P_population(X <= U) = Φ((U - μ) / σ)
```

令：

```text
zp = Φ^-1(p)
```

则条件等价于：

```text
(U - μ) / σ >= zp
```

代入 `U = x̄ + k s`：

```text
(x̄ - μ) / σ + k(s / σ) >= zp
```

设：

```text
Z = √n (x̄ - μ) / σ ~ N(0,1)
V = (n - 1)s² / σ² ~ χ²(n-1)
ν = n - 1
```

则：

```text
(x̄ - μ) / σ = Z / √n
s / σ = √(V / ν)
```

覆盖条件变成：

```text
Z / √n + k√(V / ν) >= zp
```

两边乘以 `√n`：

```text
Z + k√n √(V / ν) >= zp√n
```

整理为：

```text
(zp√n - Z) / √(V / ν) <= k√n
```

左边服从非中心 t 分布：

```text
T ~ t_ν(δ)
ν = n - 1
δ = zp√n
```

因此选择 `k` 满足：

```text
P(T <= k√n) = γ
```

也就是：

```text
k = t_{ν,δ;γ} / √n
```

其中 `t_{ν,δ;γ}` 是自由度 `ν`、非中心参数 `δ` 的非中心 t 分布的 `γ` 分位数。

单侧下容限限的推导完全对称。

## 4. 正态双侧容限区间的推导思路

双侧区间为：

```text
[x̄ - k s, x̄ + k s]
```

要求：

```text
P_sample{
  Φ((x̄ + ks - μ) / σ) - Φ((x̄ - ks - μ) / σ) >= p
} = γ
```

这里同时涉及 `x̄` 和 `s` 的抽样分布。与单侧情形相比，双侧 `k` 通常没有简单闭式表达式，需要通过数值积分、表格或近似公式求解。

常见方法包括：

- Howe 近似法。
- Wald-Wolfowitz 近似法。
- Owen 方法。
- 非中心 t 或相关分布的数值解法。
- 软件中的 exact 或 near-exact 算法。

ISO 附录表格、NIST、SAS、R `tolerance` 包、NCSS/PASS 等软件系统都基于这些理论和数值算法给出 `k` 因子。

## 5. 分布无关法的精确推导

设总体分布函数为 `F`，只要求连续。由概率积分变换：

```text
Ui = F(Xi) ~ Uniform(0,1)
```

排序后：

```text
U(1) <= U(2) <= ... <= U(n)
```

且：

```text
F[X(i)] = U(i)
```

### 5.1 单侧上容限限

取：

```text
U_limit = X(s)
```

该上限覆盖至少 `p` 比例总体的条件是：

```text
F[X(s)] >= p
```

也就是：

```text
U(s) >= p
```

由于 `U(s)` 是均匀分布样本的第 `s` 个阶次统计量：

```text
P{U(s) >= p}
= Σ_{j=0}^{s-1} C(n,j) p^j (1-p)^(n-j)
```

所以选择最小的 `s`，使：

```text
Σ_{j=0}^{s-1} C(n,j) p^j (1-p)^(n-j) >= γ
```

若直接取样本最大值 `X(n)` 为上限，则：

```text
P{F[X(n)] >= p} = 1 - p^n
```

因此要达到置信度 `γ`：

```text
1 - p^n >= γ
```

即：

```text
n >= ln(1 - γ) / ln(p)
```

例子：

```text
p = 0.95
γ = 0.95
n >= ln(0.05) / ln(0.95) ≈ 58.4
```

所以至少需要：

```text
n = 59
```

### 5.2 双侧分布无关区间

取：

```text
[X(r), X(s)]
```

该区间覆盖的总体比例为：

```text
F[X(s)] - F[X(r)] = U(s) - U(r)
```

对于均匀分布阶次统计量：

```text
U(s) - U(r) ~ Beta(s - r, n - s + r + 1)
```

因此：

```text
P{U(s) - U(r) >= p}
= 1 - I_p(s - r, n - s + r + 1)
```

其中 `I_p(a,b)` 是正则化不完全 beta 函数。

选择 `r`、`s`，使：

```text
1 - I_p(s - r, n - s + r + 1) >= γ
```

特别地，若使用最小值和最大值：

```text
[X(1), X(n)]
```

则覆盖概率为：

```text
γ = 1 - n p^(n-1) + (n - 1)p^n
```

这个公式常用于估算双侧非参数容限区间需要多大的样本量。

## 6. 与其它区间的区别

### 6.1 置信区间

置信区间通常覆盖总体参数，例如：

```text
μ
σ
比例参数
```

它回答的是：

```text
参数在哪里？
```

### 6.2 预测区间

预测区间通常覆盖一个或若干个未来观测值。

它回答的是：

```text
下一个样本值可能在哪里？
```

### 6.3 容限区间

容限区间覆盖总体中至少某个比例的个体。

它回答的是：

```text
总体中至少 p 比例的个体会落在哪里？
```

这就是为什么质量工程、验证、放行、材料许用值中经常用容限区间，而不是均值置信区间。

## 7. 主要文献

经典文献和资料线索如下：

1. Wilks, S. S.  
   关于 distribution-free tolerance limits 的早期工作，奠定了基于阶次统计量的非参数容限区间方法。很多资料把非参数容限区间称作 Wilks 方法。

2. Wald, A. and Wolfowitz, J.  
   对非参数容限限和样本量选择有早期系统结果。

3. Owen, D. B.  
   对正态容限限、非中心 t 分布、双侧正态容限区间和容限因子表格有重要贡献。

4. Natrella, M. G.  
   NBS Handbook 91 / NIST 工程统计手册体系中的工程化公式和表格来源之一。

5. Hahn, G. J., Meeker, W. Q., and Escobar, L. A.  
   *Statistical Intervals: A Guide for Practitioners and Researchers*. Wiley.  
   这是实践界很常用的统计区间教材，覆盖置信区间、预测区间、容限区间等。

6. Krishnamoorthy, K. and Mathew, T.  
   *Statistical Tolerance Regions: Theory, Applications, and Computation*.  
   这是现代容限区间和容限区域理论、应用、计算的系统教材。

## 8. 其它使用场景

### 8.1 制造业与质量工程

容限区间用于比较过程自然波动与规格限，例如：

```text
统计容限限是否落在工程规格限以内？
```

如果：

```text
[L_tolerance, U_tolerance] ⊆ [L_spec, U_spec]
```

则可说明在给定置信度下，至少 `p` 比例的过程输出满足规格。

### 8.2 医疗器械验证

医疗器械设计验证和过程验证中，经常需要证明关键质量特性在安全或性能限值内。容限区间可用于表达：

```text
以 γ 置信度，至少 p 比例产品满足设计要求
```

### 8.3 制药工艺验证与批放行

制药中常见用途包括：

- 工艺验证 acceptance criteria。
- 含量均匀度。
- 分析方法验证。
- API 或制剂关键质量属性。
- 放行和货架期规格限制定。

例如：

```text
以 95% 置信度，至少 99% 批次或单位落在 85% 到 115% label claim 内
```

### 8.4 药典和分析方法验证

USP <1210> 等统计工具指南中提到双侧容限区间可用于同时评价 accuracy 和 precision，因为它直接关注个体测量结果是否落入预设可接受区间。

### 8.5 航空航天材料许用值

复合材料、金属材料等材料许用值中常见：

```text
A-basis: 99% population / 95% confidence
B-basis: 90% population / 95% confidence
```

这本质上是单侧下容限限，用来给出保守的材料强度许用值。

### 8.6 统计软件

很多统计软件实现了容限区间：

- NIST Dataplot。
- SAS/QC。
- R `tolerance` 包。
- NCSS/PASS。
- Minitab、JMP 等质量工程软件。

## 10. 一句话总结

ISO 16269-6 的核心过程是：在正态假设下用 `x̄ ± ks` 和容限因子 `k`，或在无分布假设下用排序样本的阶次统计量，来构造一个以置信度 `γ` 覆盖总体中至少 `p` 比例个体的统计区间。
